# Installation

1. run :

```bash
git clone https://github.com/mkalbore/wportfolio.git
```

to clone my repo

2. run

```bash
npm install
```

to install all dependencies

3. run

```bash
npm run dev
```

to run server

4. go to http://localhost:3000/ to view project

If u have questions ❓, feel free to ask.
